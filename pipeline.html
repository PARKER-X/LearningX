Introduction

  • “NLP understands language.” ❌
  • “Transformers read text.” ❌
  • “LLMs know what words mean.” ❌

NLP is a pipeline of transformations: text → numbers → decisions.

Part 1: The Core Problem

Computers see: 

"I", " ", "l", "o", "v", "e", " ", "c", "a", "t", "s"

There is no meaning. Only symbols.

Part 2: Tokenization

Breaking text into atomic pieces:

"I love neural networks"

Word-based:    ["I", "love", "neural", "networks"]
Subword-based: ["I", "love", "neur", "al", "network", "s"]
Character:     ["I"," ","l","o","v","e",...]

Subwords solve unknown words + vocabulary explosion.

Part 3: Vocabulary & IDs

Words → integers:

"I"        → 17
"love"     → 2031
"neural"   → 9124
"networks" → 4512

Sentence → [17, 2031, 9124, 4512]

Zero meaning yet — just numeric IDs.

Part 4: Embeddings

Words → high-dimensional vectors:

"cat" → [0.21, -1.03, 0.77, ..., 0.12]

Similar words → close in vector space.

Embedding Layer = Lookup Table

[17, 2031, 9124, 4512]
↓
[
  v("I"),
  v("love"),
  v("neural"),
  v("networks")
]

Part 5: Position Matters

Order of words changes meaning:

  • "Dog bites man" ≠ "Man bites dog"

Add positional embeddings:

final_embedding = word_embedding + position_embedding

Part 6: Feature Extraction

Pipeline so far:

Text → Tokens → IDs → Embeddings → Position-aware vectors

Old NLP: handcrafted features (POS, NER, dependency parsing)

Modern NLP: end-to-end learning via self-attention.

Part 7: Attention

Each word asks:

"Which other words should I pay attention to?" 
Attention(Q,K,V) = softmax(QKᵀ / √d) V

Part 8: Transformers

  • Stacked self-attention + feedforward layers
  • Residual connections & normalization
  • Early layers: syntax
  • Middle layers: entities & relations
  • Late layers: intent, topic, reasoning

Part 9: Task Heads

  • Sentiment: [CLS] → Linear → Positive/Negative
  • NER: Token embeddings → Label per token
  • Translation: Encoder → Decoder → Next token
  • Language Modeling: Predict next token

Part 10: Training

FOR each batch:
  Forward pass
  Compute loss
  Backpropagate
  Update weights

Example (language modeling): input: "I love neural" → target: "networks"

Part 11: Inference

User text
↓
Tokenize
↓
Embed
↓
Transformer layers
↓
Output probabilities
↓

↓
Repeat
